# Phase 1

Phase 1 opens at:

```text
https://gsmg.io/theseedisplanted
```

The page is mostly images, with no obvious next step.

## Hidden Form

Looking at the page source shows a hidden password form:

```text
phase1verification
```

So this phase is asking for a password.

## Image Clues

The images can be rearranged into fragments:

```text
warning
crypto
logic
can you
dig it
+ -
```

Searching the useful parts, especially without `crypto`, points to the song:

```text
The Warning - Logic
```

## Lyric Match

The page URL, `theseedisplanted`, appears in the lyric line:

```text
The seed is planted when opposites attract
```

Other page fragments also match the lyrics:

```text
Can you dig it?
Phase one
Phase two
The flower blossoms through what seems to be a concrete surface
This is the warning
```

## Password

Trying lyric lines as lowercase text with spaces removed gives the password:

```text
theflowerblossomsthroughwhatseemstobeaconcretesurface
```

Submitting it redirects to Phase 2:

```text
https://gsmg.io/choiceisanillusioncreatedbetweenthosewithpowerandthosewithoutaveryspecialdessertiwroteitmyself
```

## Notes

This phase teaches two useful puzzle rules:

```text
hints may need outside references
passwords and URLs are lowercase with no spaces
```
